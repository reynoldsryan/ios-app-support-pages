# Ball Machine Practice Support

Ball Machine Practice is a local-first iPhone app for planning tennis and pickleball ball-machine practice queues, running courtside drill sessions, and saving the one adjustment you want to remember next time.

## Getting Help

For support, bug reports, privacy questions, or feature requests, open an issue here:

https://github.com/reynoldsryan/ios-app-support-pages/issues

Please include:

- app name: Ball Machine Practice
- iPhone model
- iOS version
- app version
- a short description of what happened

Do not include payment details, home addresses, court access codes, private club/member information, health information, photos of minors, or anything you do not want public.

## Important Boundary

Ball Machine Practice is a practice planning and local note-taking app. It does not connect to, pair with, control, diagnose, calibrate, or operate any ball machine. It does not provide coaching, medical, injury-prevention, safety, or equipment-maintenance advice.
