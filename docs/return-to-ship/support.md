# Return to Ship Support

Return to Ship is a local-first iPhone app that helps cruise passengers build one clear return plan for a port day. It uses manually entered all-aboard, buffer, last-tender, pier, ship-time, and transit notes to show when to head back and which reboarding checklist items remain.

## Getting Help

For support, bug reports, privacy questions, or feature requests, open an issue here:

https://github.com/reynoldsryan/ios-app-support-pages/issues

Please include:

- app name: Return to Ship
- iPhone model
- iOS version
- app version
- a short description of what happened

Do not include passport numbers, cruise booking numbers, cabin numbers, payment information, exact travel documents, or anything you do not want public.

## Important Boundary

Return to Ship is a planning utility and checklist. It does not use live maps, routing, traffic, tender feeds, weather, port-status feeds, cruise-line systems, itinerary import, account integrations, OCR, AI, or official ship data. It cannot guarantee that you will make all-aboard and does not replace posted ship instructions, crew directions, official time-zone/ship-time guidance, port authority instructions, or your cruise line's policies.
