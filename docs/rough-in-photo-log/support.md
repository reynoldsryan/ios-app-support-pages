# Rough-In Photo Log Support

Rough-In Photo Log helps contractors create local rough-in photo records and proof-pack PDFs from job-site photos, fixture/room tags, and notes.

## Getting Help

For support, bug reports, privacy questions, or feature requests, open an issue here:

https://github.com/reynoldsryan/ios-app-support-pages/issues

Please include:

- app name: Rough-In Photo Log
- iPhone model
- iOS version
- app version
- a short description of what happened

Do not include private job details, customer information, addresses, invoices, or photos in public issues.
