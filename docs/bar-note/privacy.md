# Bar Note Privacy Policy

Effective date: 2026-05-13

Bar Note is a local-first iOS app for saving user-created practice notes about music lessons.

## Data collection

Bar Note does not collect personal data from this app.

## Local app content

Practice pieces, marked bars, teacher corrections, practice history, and export text are intended to remain on the user's device. Bar Note does not provide a cloud account, teacher account, social sharing service, sheet-music library, audio recognition, or AI/OCR sheet-music extraction service.

## Photos and camera

If the app requests camera or photo access, it is used only so the user can add their own practice snippet/reference inside the app. Bar Note does not upload those images to a server.

## Purchases

Bar Note may offer a one-time Pro unlock through Apple's in-app purchase system. Purchase processing is handled by Apple. Bar Note uses purchase entitlement status only to unlock paid features in the app.

## Contact

For privacy questions, use the support contact associated with this App Store listing.
