# Bar Note Support

Bar Note helps music students save the exact bar or measure their teacher asked them to fix, attach a correction note, and run focused practice reps. Everything is designed for local, user-created practice notes.

## Getting help

If something is not working, include:

- the device model and iOS version;
- what you were trying to do;
- whether the issue happened while creating a piece, marking a bar, practicing reps, restoring purchase, or exporting a summary;
- any visible error message.

Contact: use the support link or email associated with this App Store listing.

## Data and privacy

Bar Note is local-first. It does not include sheet-music recognition, audio analysis, teacher accounts, cloud sync, or a public music library.

See the privacy policy: [Bar Note Privacy Policy](./privacy.md)
