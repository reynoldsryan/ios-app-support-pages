# Separate Ticket Check Support

Separate Ticket Check is a local-first iPhone app that helps travelers evaluate risky same-day flights booked on separate tickets. It uses manually entered arrival/departure times and plain-language friction flags to produce a conservative red/yellow/green transfer-risk verdict and checklist.

## Getting Help

For support, bug reports, privacy questions, or feature requests, open an issue here:

https://github.com/reynoldsryan/ios-app-support-pages/issues

Please include:

- app name: Separate Ticket Check
- iPhone model
- iOS version
- app version
- a short description of what happened

Do not include passport numbers, ticket numbers, confirmation codes, payment information, exact travel documents, or anything you do not want public.

## Important Boundary

Separate Ticket Check is a planning utility and conservative checklist. It does not use live flight data, gate data, airline-policy feeds, airport minimum-connection-time databases, delay feeds, travel insurance systems, booking systems, maps, or account integrations. It cannot guarantee that a connection will succeed and does not replace airline, airport, immigration, customs, baggage, visa, or travel-advisory guidance.
