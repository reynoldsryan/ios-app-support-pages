# Class Pack Guard Support

Class Pack Guard is a local-first iPhone app that helps boutique fitness class-pack users manually track paid credits, expiration dates, booking windows, late-cancel cutoffs, and next actions before credits or fees are lost.

## Getting Help

For support, bug reports, privacy questions, or feature requests, open an issue here:

https://github.com/reynoldsryan/ios-app-support-pages/issues

Please include:

- app name: Class Pack Guard
- iPhone model
- iOS version
- app version
- a short description of what happened

Do not include private class schedules, studio account details, payment information, screenshots with personal information, or any information from a studio app that you do not want public.

## Important Boundary

Class Pack Guard is a manual tracker. It does not sync with studios, ClassPass, Mindbody, calendars, email, or booking systems. It does not book or cancel classes and does not verify studio policies. Always confirm important dates and cancellation rules in your studio app.
