# Breaker Map Support

Breaker Map is a local-first iPhone utility for keeping a simple map of breaker numbers, rooms, and important devices.

## Getting help

If you need support, include:

- The app name: Breaker Map
- Your iOS version and device model
- What you were trying to do
- Any steps that reproduce the issue

Contact: use the support contact listed on Ryan Reynolds' App Store developer page.

## Data and privacy

Breaker Map is designed for local home-reference notes. It does not require an account or backend service for v1.
