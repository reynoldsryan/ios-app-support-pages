# Device Fault Log Support

Device Fault Log is a local-first iPhone app for saving household device fault notes, status, photos/manual references, and support-call context in one place.

## Getting Help

For support, bug reports, privacy questions, or feature requests, open an issue here:

https://github.com/reynoldsryan/ios-app-support-pages/issues

Please include:

- app name: Device Fault Log
- iPhone model
- iOS version
- app version
- a short description of what happened

Do not include payment details, home addresses, phone numbers, warranty account credentials, serial numbers, device photos that reveal private spaces, or anything you do not want public.

## Important Boundary

Device Fault Log is an evidence and history organizer. It does not diagnose devices, provide repair instructions, identify fault codes, search manuals, contact manufacturers, control hardware, monitor networks, provide safety advice, or verify warranty/repair decisions.
