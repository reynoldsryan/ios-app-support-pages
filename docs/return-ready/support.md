# Return Ready Support

Return Ready is a local-first iPhone app that helps you track retailer return deadlines, label/packing steps, car drop-off batches, and refund value at risk.

## Getting Help

For support, bug reports, privacy questions, or feature requests, open an issue here:

https://github.com/reynoldsryan/ios-app-support-pages/issues

Please include:

- app name: Return Ready
- iPhone model
- iOS version
- app version
- a short description of what happened

Do not include order numbers, shipping labels, tracking numbers, payment information, account credentials, home addresses, or anything you do not want public.

## Important Boundary

Return Ready is a planning utility for manually entered return tasks. It does not verify retailer policies, refund eligibility, return windows, carrier availability, store hours, drop-off locations, QR codes, or label validity. Always confirm the current retailer and carrier instructions before shipping or dropping off a return.
