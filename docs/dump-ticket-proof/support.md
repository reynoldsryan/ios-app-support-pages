# Dump Ticket Proof Support

Dump Ticket Proof helps contractors create local disposal-fee proof records and shareable PDFs from dump tickets, load photos, disposal location details, and job tags.

## Getting Help

For support, bug reports, privacy questions, or feature requests, open an issue here:

https://github.com/reynoldsryan/ios-app-support-pages/issues

Please include:

- app name: Dump Ticket Proof
- iPhone model
- iOS version
- app version
- a short description of what happened

Do not include private job details, customer information, addresses, invoices, tickets, receipts, or photos in public issues.
